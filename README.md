## Getting Started

#### Installation

1. Clone the Repo

```
git clone https://github.com/mnxmnz/Svelte-Practice.git
```

```
cd Svelte-Practice
```

2. Install Project Packages

```
npm i
```

3. Run the Project (Dev Mode)

```
npm run dev

http://localhost:8080/
```
